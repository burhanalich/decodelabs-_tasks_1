## Task 1: My Tasks List

### Description
This program demonstrates how to:

- Create an empty list
- Add items using the `append()` method
- Iterate through the list using a `for` loop
- Print each item

### Code

```python
my_tasks = []

my_tasks.append("Buy Milk")
my_tasks.append("Buy bread")
my_tasks.append("Buy eggs")

for task in my_tasks:
    print(task)
```

### Output

```
Buy Milk
Buy bread
Buy eggs
```

- Python Lists
- `append()` Method
- `for` Loop
- Variables
- Printing Output
